KINOVAAPIUSBCOMMANDLAYER_API int SetTorqueFeedVelocity ( float  Command[COMMAND_SIZE])

This function sets the amount of velocity feedforward (back EMF) sent to the actuators. The parameters will reset to default values every time the robotic arm is rebooted.

Parameters
CommandThe velocity feed for each actuator.
Warning
These functions should only be used by advanced users. The user should be very cautious and be ready to turn the robot off if a problem occurs. The user should quit torque control mode and be in Trajectory-Position control mode before using theses functions.
