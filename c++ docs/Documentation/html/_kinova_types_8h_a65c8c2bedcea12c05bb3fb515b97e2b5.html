enum CONTROL_TYPE

This represents a type of control. For now, there is 3 type of control, the cartesian control, the angular control or the force control.

Enumerator
CONTROL_TYPE_CARTESIAN 

Cartesian control. (translation and orientation)

CONTROL_TYPE_ANGULAR 

Angular control. (joint by joint)

CONTROL_TYPE_UNKNOWN 

Unknown control type.
