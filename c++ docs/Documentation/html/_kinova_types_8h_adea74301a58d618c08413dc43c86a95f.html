enum PERIPHERAL_TYPE

That represents the type of a peripheral.

Enumerator
PERIPHERAL_TYPE_NONE 

Unknown type.

PERIPHERAL_TYPE_ANY 

Abstract peripheral. internal use only.

PERIPHERAL_TYPE_UNKNOWN 

Unknown peripheral. internal use only.

PERIPHERAL_TYPE_ACTUATOR_GENERIC 

A generic rotary actuator.

PERIPHERAL_TYPE_ACTUATOR_BIG_19NM 

A big 19 Newton*Meter rotary actuator.

PERIPHERAL_TYPE_ACTUATOR_BIG_37NM 

A big 37 Newton*Meter rotary actuator.

PERIPHERAL_TYPE_ACTUATOR_SMALL_7NM 

A small 7 Newton*Meter rotary actuator.

PERIPHERAL_TYPE_LINEAR_ACTUATOR_GENERIC 

A generic linear actuator.

PERIPHERAL_TYPE_LINEAR_ACTUATOR_120N 

A 120 Newton(finger) linear actuator.

PERIPHERAL_TYPE_JOYSTICK 

A generic joystick.

PERIPHERAL_TYPE_VIRTUAL_JOYSTICK 

A virtual joystick. This is mainly used by the API.

PERIPHERAL_TYPE_KINOVA_JOYSTICK_3AXIS 

A kinovian joystick.

PERIPHERAL_TYPE_UNIVERSAL_INTERFACE_V2 

A universal interface V2.

PERIPHERAL_TYPE_CAN_INTERFACE 

A CAN interface on the main board.
