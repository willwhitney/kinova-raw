KINOVAAPIUSBCOMMANDLAYER_API int Ethernet_InitAPI ( void  )

This function sets a default ethernet configuration and tries to initialize the API. It is the first function you call if you want the rest of the library.

Returns
  • NO_ERROR_KINOVA if operation is a success.
  • ERROR_LOAD_COMM_DLL if unable to load the communication layer.
  • ERROR_INIT_COMM_METHOD if unable to load the Ethernet_Communication_InitCommunicationEthernet function from communication layer.
  • ERROR_CLOSE_METHOD if unable to load the Ethernet_Communication_CloseCommunication function from communication layer.
  • ERROR_GET_DEVICE_COUNT_METHOD if unable to load the Ethernet_Communication_GetDeviceCount function from communication layer.
  • ERROR_SEND_PACKET_METHOD if unable to load the either the Ethernet_Communication_SendPacketList, Ethernet_Communication_SendPacket or Ethernet_Communication_BroadcastSendPacket function from communication layer.
  • ERROR_SET_ACTIVE_DEVICE_METHOD if unable to load the Ethernet_Communication_SetActiveDevice function from communication layer.
  • ERROR_GET_DEVICES_LIST_METHOD if unable to load the Ethernet_Communication_GetDevices function from communication layer.
  • ERROR_GET_ACTIVE_DEVICE_METHOD if unable to load the Ethernet_Communication_GetActiveDevice function from the communication layer.
  • ERROR_INIT_DATA_STRUCTURES_METHOD if unable to load the Ethernet_Communication_InitDataStructures function from the communication layer.
  • ERROR_OPEN_RS485_ACTIVATE if unable to load the Ethernet_Communication_OpenRS485_Activate function from the communication layer.
  • ERROR_SCAN_FOR_NEW_DEVICE if unable to load the Ethernet_Communication_ScanForNewDevice function from the communication layer.
